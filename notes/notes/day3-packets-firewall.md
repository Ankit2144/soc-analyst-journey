# Day 3 - Packets, Firewall, VPN, and NAT

## What is a packet?
A packet is a small piece of data sent across a network.

## What is a frame?
A frame is data used at the local network level. It uses MAC addresses.

## What is TCP?
TCP is a reliable protocol. It checks that data is delivered correctly.

## TCP 3-way handshake
TCP connection starts with three steps:

1. SYN
2. SYN-ACK
3. ACK

This confirms that both devices are ready to communicate.

## What is a firewall?
A firewall allows or blocks network traffic based on rules.

## What is VPN?
A VPN creates an encrypted tunnel between your device and another network.

## What is NAT?
NAT allows many private IP devices to share one public IP address.

## What is port forwarding?
Port forwarding sends traffic from one public port to a device inside a private network.

## Why this matters for SOC?
SOC analysts must understand network traffic, suspicious connections, firewall alerts, and VPN login activity.
