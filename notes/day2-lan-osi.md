# Day 2 - LAN and OSI Model

## What is LAN?
LAN means Local Area Network. It is a small network inside a home, office, school, or building.

## What is a switch?
A switch connects devices inside the same local network.

## What is a router?
A router connects different networks together. Example: your home network to the internet.

## What is DHCP?
DHCP automatically gives IP addresses to devices.

## What is ARP?
ARP finds the MAC address of a device using its IP address.

## What is the OSI model?
The OSI model is a 7-layer model that explains how data moves through a network.

## OSI 7 layers
1. Physical - cables, signals, Wi-Fi
2. Data Link - MAC address, switches
3. Network - IP address, routers
4. Transport - TCP and UDP
5. Session - starts and manages sessions
6. Presentation - data format and encryption
7. Application - apps like browser, email, DNS, HTTP

## TCP vs UDP
TCP is reliable and checks delivery.
UDP is faster but does not check delivery.

## What is a port?
A port identifies a service running on a device.

## Common ports
- 80 = HTTP
- 443 = HTTPS
- 22 = SSH
- 53 = DNS
- 25 = SMTP
- 3389 = RDP
