# Day 1 - Networking Basics

## What is a network?
A network is two or more devices connected together to share data.

## What is the internet?
The internet is a huge global network of many smaller networks.

## What is an IP address?
An IP address is a number that identifies a device on a network.

## Public IP vs Private IP
A public IP is used on the internet.  
A private IP is used inside a home or office network.

## What is a MAC address?
A MAC address is a unique hardware address of a network device.

## What is ping?
Ping is used to check if another device is reachable on the network.

## What is a packet?
A packet is a small piece of data sent across a network.
